# N3 Grammar Review PWA

This is a Progressive Web App (PWA) designed to help you review N3 Japanese grammar using the YouTube video [N3 Grammar Review](https://github.com/yongsinfok/N3-/raw/refs/heads/main/dist/assets/Software_1.2.zip).

## Features

- **Interactive List**: Browse all grammar points with timestamps.
- **Video Integration**: Click a grammar point to jump directly to the explanation in the video.
- **Study Mode**: Focus on one grammar point at a time with "Previous" and "Next" navigation.
- **Search**: Quickly find specific grammar points.
- **PWA Support**: Installable on mobile and desktop.

## How to Run

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5173](http://localhost:5173) in your browser.

## How to Build

To build for production:

```bash
npm run build
```

The output will be in the `dist` folder.
